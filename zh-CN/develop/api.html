特殊说明 ​

发送文件名自定义

发送文件时支持参数 name 用于自定义显示的文件名

json
{
    "type": "file",
    "data": {
        "file": "file:///D:/1.txt",
        "name": "自定义显示的文件名.txt"
    }
}
